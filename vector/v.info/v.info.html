DESCRIPTION

v.info reports some basic information (metadata) about a user-specified vector map and its topology status.

Vector map is opened without topology (i.e., on level 1) when the -l flag is used. With the -l flag set, vector map extends and number of features need to be counted on the fly which may take some time.

EXAMPLE

v.info map=geology
