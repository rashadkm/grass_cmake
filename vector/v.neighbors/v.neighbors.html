DESCRIPTION

Makes each cell value a function of the attribute values assigned to the vector points or centroids around it, and stores new cell values in an output raster map layer.

EXAMPLE

Count the number of schools for a given grid (North Carolina sample dataset) in a moving windows: 
g.region rast=zipcodes -p res=500
v.neighbors input=schools_wake output=schools_wake_500m method=count size=3
