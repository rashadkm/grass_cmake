DESCRIPTION

r.series.interpol interpolates raster maps which are located temporal or spatial in between existing input raster maps. The interpolation is performed at specific sampling positions. The sampling position for each output map must be specified. Sampling positions are always located in the normalized intervall (0;1). The following interpolation methods are supported.
  • linear: Linear interpolation. Two input maps are required. The first input map is located at position 0.0, the second input map at position 1.0.

EXAMPLES

Interpolate linear three new maps at 3 sampling positions in between the interval (0;1) First prepare the input maps: 

g.region s=0 n=80 w=0 e=120 b=0 t=50 res=10 res3=10 -p3

r.mapcalc --o expr="prec_1 = 100"
r.mapcalc --o expr="prec_5 = 500"

Interpolate 

r.series.interpol --o --v input=prec_1,prec_5 output=prec_2,prec_3,prec_4 sampoint=0.25,0.5,0.75 method=linear

Interpolate using the file option: First prepare the input file: 

TMP_FILE=`g.tempfile pid=1`
cat > "${TMP_FILE}" << EOF
prec_2|0.25
prec_3|0.5
prec_4|0.75
EOF

Interpolate 

r.series.interpol --o --v input=prec_1,prec_5 file="${TMP_FILE}" method=linear
The created maps will have the values 200, 300 and 400.
