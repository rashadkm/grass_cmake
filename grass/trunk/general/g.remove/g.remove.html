DESCRIPTION

g.remove removes data files matching a pattern given by wildcards or POSIX Extended Regular Expressions. If the -f force flag is not given then nothing is removed, instead the list of selected file names is printed to stdout as a preview of the files to be deleted.

EXAMPLES

Delete all raster maps starting with "tmp_" in the current mapset: 
# show matching raster maps but do not delete yet (as verification)
g.remove type=rast pattern="tmp_*"

# actually delete the matching raster maps
g.remove -f type=rast pattern="tmp_*"
Delete all raster maps starting with "stream_" in the current mapset, but exclude those ending with "_final": 
g.remove -f type=rast pattern="stream_*" exclude="*_final"
