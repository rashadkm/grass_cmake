DESCRIPTION

v.to.rast3 converts a GRASS 3D vector point map to a GRASS 3dgrid raster3d (voxel) map.

NOTES

When converting from a 3D vector point layer to a raster3d map a vector point is converted into a single 3d cell representing the location of the vector point. As 3d cell value the database attribute of the vector point is stored as float.
