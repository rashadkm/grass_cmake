NAME

r.carve - carve a vector stream into an elevation model
(2)

SYNOPSIS

r.carve [-q] [-n] rast=name vect=name output=name [site=name] [width=value] [depth=value]

DESCRIPTION

r.carve

The program will take the vector stream data, transform them to raster, and subtracts a defaultdepth+additionaldepth from the DEM. If the given width is more than 1 cell it will carve the stream with the given width. With -n option it should eliminate all flat cells within the stream, so when and if the water gets into the stream it will flow. Site option generates x,y,z for points which define the stream with the z value of the bottom of the carved-in stream. These sites can then be combined with contour sites to interpolate a new DEM with better representation of valleys. It has not been thoroughly tested so not all option may work properly - but this was the intention.

OPTIONS

Flags:

-q
quiet - do not show progress
-n
no flat areas allowed in flow direction

Parameters:

rast=name
raster input elevation map
vect=name
vector input map containing stream
output=name
raster output map of modified elevation
site=name
site output map for adjusted stream points
width=value
width of streams (in meters)
depth=value
additional depth

NOTES

r.carve does not create a depressionless DEM because many depressions are in flat areas and not in the streams.
