GRASS GIS Reference Manual

This reference manual details the use of modules distributed with Geographic Resources Analysis Support System (GRASS), an open source (GNU GPL'ed), image processing and geographic information system (GIS).

The current command line rendering mechanism is direct rendering into a file. The driver is selected by setting the GRASS_RENDER_IMMEDIATE variable.

Display drivers:

  • CAIRO driver
  • PNG driver
  • PostScript driver
