DESCRIPTION

d.barscale displays a barscale on the graphics monitor at the given screen coordinates. If no coordinates are given it will draw the barscale in the bottom left of the display. It can draw the scale or north arrow in a number of styles.

NOTE

d.barscale will not work in Lat/Lon locations as the horizontal scale distance changes with latitude. Try d.grid instead.
